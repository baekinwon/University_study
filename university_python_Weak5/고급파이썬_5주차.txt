1 반복문

루프
- 어떤 조건에 도달할 때까지 계속하여 반복되는 일련의 명령문
- 동일한 작업을 여러 번 수행

반복문 필요성
- 프로그램의 간결화

for문
- 횟수 제어 반복
  1) 반복의 횟수를 미리 아는 경우에 사용
  2) for문, for반복문, for루프
  3) 콜론 누락 주의
  4) block을 사용

range()함수
- range() 함수 : 숫자들을 생산하는 공장
- range 뜻 : 범위, 영역, 사거리
- 사용법
  1) start 값 생략시, 0이  기본 값으로 설정
  2) step 값 생략 시, 1이 기본 값으로 설정
  3) stop 값은 생략 불가능하며, 출력에 포함되지 않는 수
  4) step 값에 음수도 가능
  5) print()의 end 변수 설정을 통해 줄바꿈 변경 가능

while문
- 어떤 조건이 만족되는 동안 반복
- 반복의 횟수는 모르지만, 반복의 조건은 알고 있는 경우 사용

중첩반복문
- 반복문 내부에 다른 반복문이 포함된 형태
- 외부 반복문과 내부 반복문으로 나누어 짐

무한 반복
- 반복이 무한히 발생하는 것 = 무한 루프
- 특정 조건에서 그 무한 반복을 멈추는 것을 생각해줘야 함

continue문
- 해당 차례를 건너뛰고 반복문을 다시 수행
